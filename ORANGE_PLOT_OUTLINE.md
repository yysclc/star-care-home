# 橙橙剧情线路整合大纲

> 本文档先定义橙橙线的结构原则，不急于细写每周剧情。
>
> 核心修正：**一周不是一个地点、一个按钮、一段剧情。**
> 每一周应被设计成一个由多个入口、条件分支、补救机会和延迟反馈组成的事件网络。

---

## 1. 橙橙线的基础理念

橙橙线不是“每周点击一次橙橙按钮，播放一段剧情”的角色线。它应该让玩家在照护所的日常操作中逐步发现她、理解她，并意识到自己的前序选择会改变后续能看见什么、能做什么、能补救什么。

每一周可以有不同的结构。有的周是探索链，有的周是家长沟通链，有的周是危机处理链，有的周是机构压力链。周与周之间不必复用同一套流程。

### 橙橙人设底盘

橙橙的核心不是“喜欢画画”，而是：

- **声音敏感**：对高频、突兀、不可预测的声音更容易紧张或退缩。
- **视觉表达优势**：她通过画作记录、整理和表达世界。
- **低口语沟通**：她可能用单字、手势、推画、收画、停笔、转身等方式表达。
- **秩序与可预测性需求**：画笔排列、纸张位置、桌面秩序、他人靠近方式都会影响她的参与状态。
- **绘画材料感官偏好**：光滑纸张、柔软布料、颜料触感、纸张摩擦声、桌面混乱程度都可以成为剧情和玩法变量。

因此，橙橙的画不是“天赋展示品”，而是她的沟通方式、情绪记录、选择权表达和剧情证据。

## 2. 四条线路如何整合

四条线不是互斥路线，而是同一条橙橙主线中的不同功能层。

| 线路 | 功能 | 在结构中的位置 |
|------|------|----------------|
| 理解与连接线 | 主轴，推动玩家从“靠近”到“理解” | 每周都存在，但表现方式不同 |
| 边界与尊严线 | 中期伦理冲突，考验玩家是否保护橙橙的选择权 | 在公开展示、拍摄、家长压力等节点中爆发 |
| 家庭修复线 | 情感高潮，处理父亲焦虑与亲子误解 | 通过家长沟通、父亲来访、转院风险推进 |
| 画作叙事线 | 贯穿机制，用画作承载线索、证据和结局拼图 | 每周可出现，但不一定每周都“获得作品” |

推荐整合方式：

```
理解与连接线 = 主轴
画作叙事线 = 贯穿工具
家庭修复线 = 后期情感压力
边界与尊严线 = 中期价值冲突
```

## 3. “周”的正确设计单位

一周应被视为一个**事件网络**，而不是单个剧情节点。

### 一周可以包含的入口

同一周内，橙橙相关内容可以从多个位置触发：

- 房间中的橙橙按钮
- 房间中的普通自由行动按钮
- 办公室所长聊天或请教
- 电脑中的建设、家长沟通、消息回复
- 固定活动后的插入事件
- 自由活动结束后的特殊剧情
- 周结算时的观察记录或画作反馈

不是每周都必须包含以上所有入口。关键是：**每周应根据主题选择不同的入口组合**，让玩家感觉自己在照护所里行动，而不是在读固定章节。

### 一周可以包含的节点类型

| 节点类型 | 作用 |
|----------|------|
| 主入口事件 | 本周第一个橙橙相关触发点，通常用于提出问题 |
| 复盘事件 | 通过所长、同事或记录系统解释玩家刚才看到的现象 |
| 补救事件 | 玩家早期选错后，提供一次学习后修正的机会 |
| 条件后续事件 | 根据前序选择、建设状态、好感或资源触发不同内容 |
| 资源/建设事件 | 让玩家通过经营行动改变后续剧情条件 |
| 家长沟通事件 | 将橙橙的状态转化为对外沟通压力 |
| 自由活动后事件 | 在玩家行动结束后呈现延迟后果或隐藏反馈 |
| 周结算反馈 | 总结玩家本周到底“看见了什么”，而不是只总结数值 |

## 4. 每周结构必须不同

后续细写时，不要把每周写成同一种模式。以下是结构类型示例，不是最终剧情细案。

### A. 探索链结构

适合建立新关系、新机制、新空间。

结构示例：

```
房间初遇
  → 玩家第一次选择
  → 办公室所长复盘
  → 解锁建设或新入口
  → 根据是否建设/是否选对触发不同后续
  → 周结算反馈观察质量
```

第一周可以使用这种结构：活动室见到橙橙，不同选择影响所长复盘；聊完所长后解锁绘画室；是否建成绘画室决定本周后续能否进入第三段互动。

### B. 家长沟通链结构

适合把橙橙的表达转化为外部压力。

结构示例：

```
绘画室橙橙事件
  → 玩家观察/记录画作
  → 电脑或办公室触发家长沟通
  → 父亲根据玩家表达方式产生不同态度
  → 自由活动结束后出现橙橙的延迟反馈
```

第二周可以使用这种结构：直接进入绘画室，橙橙画画；玩家如何记录画作会影响之后与父亲沟通；自由活动结束后再用特殊剧情反馈橙橙是否感到被误读或被保护。

### C. 危机处理链结构

适合声音敏感、感官过载、环境变化等事件。

结构示例：

```
突发事件
  → 玩家即时处理
  → 所长或课程解释专业原因
  → 解锁道具/支持方式
  → 后续房间互动验证玩家是否真正学会
```

### D. 边界冲突链结构

适合媒体、展示、赞助、公开活动。

结构示例：

```
机构收到外部机会
  → 机会需要使用橙橙的画作或形象
  → 玩家决定是否征询/保护/妥协
  → 机构数值获得收益或损失
  → 橙橙后续互动状态改变
```

### E. 证据整合链结构

适合后期家庭冲突或转院风险。

结构示例：

```
父亲提出强烈要求
  → 玩家整理前几周画作和记录
  → 可选择先找所长复盘
  → 与父亲谈判
  → 成功/暂缓/失败进入不同尾声
```

---

## 5. 分支不应只改变数值

橙橙线的选择后果至少应该影响以下三类内容之一：

1. **后续入口是否出现**  
   例如：是否解锁绘画室后续互动、是否出现所长复盘、是否触发父亲沟通。

2. **同一事件的文本和姿态变化**  
   例如：橙橙是继续画、收起本子、转身、递画，还是只把画留在桌上。

3. **未来周的可用证据和选项**  
   例如：玩家是否拥有某张画作、是否能用“观察记录”说服父亲、是否能向橙橙征询同意。

数值变化仍然需要保留，但不能让玩家感觉选择只是在加减好感。

## 6. 建议变量与标记

这些变量用于支撑“事件网络”，不是要求一次性全部实现。

| 变量/标记 | 用途 |
|-----------|------|
| `orange_favor` | 橙橙对主角的长期信任基础 |
| `orange_first_contact` | 第一周初遇方式：安静观察 / 直接打断 / 补救成功 |
| `orange_observation_quality` | 玩家是否记录到有效细节 |
| `painting_room_built` | 绘画室是否建成，影响橙橙后续活动空间 |
| `orange_artifacts` | 已获得或记录的关键画作 |
| `orange_boundary_respected` | 是否尊重过橙橙拒绝展示/拍摄的权利 |
| `orange_overload_support` | 是否掌握声音或感官支持方式 |
| `orange_parent_trust` | 父亲对机构和主角的信任 |
| `orange_parent_understanding` | 父亲是否开始理解橙橙的表达方式 |
| `orange_transfer_risk` | 转院风险状态 |

## 7. 每周设计模板

后续一周一周写时，建议统一使用这个模板。

### 第X周：周主题

**本周结构类型**  
说明这一周采用探索链、家长沟通链、危机处理链、边界冲突链，还是混合结构。

**本周核心问题**  
玩家这周要学会什么，或被迫面对什么。

**入口列表**  
列出本周橙橙相关内容会从哪些入口出现。不要默认只有一个橙橙按钮。

**事件网络**

```
入口A
  → 选择/观察
  → 条件分支
入口B
  → 复盘/沟通/补救
入口C
  → 根据前序状态触发不同后续
自由活动结束/周结算
  → 延迟反馈
```

**关键条件**

- 需要哪些前置变量
- 哪些分支由建设状态决定
- 哪些分支由前序选择决定
- 哪些分支由好感或画作记录决定

**后果设计**

- 立即反馈
- 延迟反馈
- 解锁/关闭的后续入口
- 获得或错过的画作/记录
- 对父亲、所长、机构数值的影响

**本周不写的内容**

明确这一周不承担什么任务，避免每周都塞满所有线路。

---

## 8. 目前先不细写的内容

在正式逐周设计前，暂不展开以下内容：

- 每周完整对白
- 每张画作的最终文案
- 每个选项的具体数值
- 第4周以后冲突的详细剧情
- 结局文本

这些内容应在“周结构”确定后再逐步填入。

## 9. 第一周详细设计：第一次靠近

**本周结构类型**  
探索链结构：活动室初遇 → 所长复盘 → 解锁绘画室建设 → 根据建设状态触发不同后续 → 周结算反馈。

**本周核心问题**  
玩家第一次学习：靠近橙橙之前，先看见她正在用什么方式维持安全感。第一周不追求"建立亲密关系"，只建立"橙橙不进一步回避主角"的最低基础。

**本周叙事目标**
- 建立橙橙"不可被简单理解"的第一印象
- 让玩家意识到常规沟通方式（对话、提问）对橙橙无效
- 引入"画作是沟通方式"的核心概念
- 建立玩家对ASD的基本认知（非治愈、需理解）

---

### 9.1 事件网络详细设计

#### 节点1：活动室初遇（主入口事件）

**触发时机**：第一周白天，玩家首次进入活动室

**场景描述**：
- 活动室角落，一个女孩不和其他同伴一起，而是坐在地上专注绘画
- 对其他孩子的喧闹无明显反应，但对突然的声音/动作有细微退缩
- 桌上：素描本、几支铅笔、排列整齐的彩色铅笔

**玩家选择**（影响好感度，且记录方式）：

| 选择 | 橙橙反应 | 观察记录 |
|------|---------|---------|
| 安静坐在不远处观察 | 继续画画，不抬头但不收本子 | 记录"橙橙对安静 presence 不排斥" |
| 直接走近搭话："你在画什么？" | 合上本子，转开身体 | 记录"直接对话会让橙橙回避" |
|
**关键设计**：
- 无论选什么，橙橙都不会主动回应
- 但不同选择会影响她"是否收起本子"——这是第一周唯一的"好感指标"（影响橙橙好感度）
- 选择后，文本最后给予提示找所长聊聊，回办公室，所长按钮触发所长复盘解锁

**记录变量**：`orange_first_contact` = `observe` / `interrupt`

---

#### 节点2：办公室所长复盘（复盘事件）

**触发时机**：初遇后，玩家进入办公室

**所长的作用**：
- 不是"教学"，而是"帮助玩家理解刚才看到了什么"
- 根据玩家初遇选择，给出不同解释

**复盘内容分支**：

**若玩家选择"安静观察"**：
> 所长："你注意到她没有收本子吧？对橙橙来说，'不收本子'已经是很大的接受了。她不是不理人，是需要用她的方式靠近。"

**若玩家选择"直接搭话"**：
> 所长："不是你做错了什么。只是我们习惯用'问'来开始，但橙橙的世界里，'问'有时候像突然的噪音。下次试试先停下来？"
这里可以加入一个选项分支，给玩家二次机会，触发橙橙观察对话环节。选对一样加好感度。

**关键设计**：
- 复盘不是"评分"，而是"解释"
- 无论初遇选什么，所长都会给出建设性反馈（避免玩家感到"选错了就完了"，给出补救）
- 复盘后解锁**绘画室建设提示**，但是如果玩家已经建设，则给出不一样的剧情。

**解锁内容**：
- 所长提到："如果有专门的空间放她的画具，她可能会更安心"
- 解锁电脑中的"绘画室建设"选项
- 解锁变量：`painting_room_unlocked` = true

---

#### 节点3：电脑建设选择（资源/建设事件）

**触发时机**：所长复盘后，玩家进入电脑界面

**选择**：
- **建设绘画室**：消耗资金，本周内建成

**建设绘画室的影响**：
- 立即解锁：绘画室空间
- 本周可进入节点4（绘画室后续）
- 机构资金减少


**暂不建设的影响**：
- 本周无法进入绘画室
- 节点4删除，且后续橙橙剧情全部中断

---

#### 节点4A：绘画室后续（条件后续事件）


**场景描述**：
- 橙橙不立刻进入绘画室，而是站在绘画室外。
  1. 检查桌面是否干净
  2. 调整纸张位置
  3. 按颜色排列画笔
  4. 坐下，停顿，看光线

**玩家观察选择**：

| 选择 | 结果 |
|------|------|
| 安静等待（不催促） | 橙橙开始画，画的是桌上的静态物体 |（好感度较多）
| 问"你要画什么？" | 橙橙放下笔，不回应 |（好感度不变，且无法获得画作物品）
| 帮忙排列画笔 | 橙橙观察玩家的排列方式，可能微调但不排斥 |（好感度较少）

**关键设计**：
- 橙橙的"检查行为"本身就是重要观察点
- 玩家如果急于"看到她画画"，会错过这个细节
- 周结算时，观察到"检查行为"的玩家会获得更高观察质量

**解锁画作**：《桌》（第一阶段）

**记录变量**：
- `orange_artifacts`：判断是否获得画作，添加《桌》



#### 节点5：周结算反馈前，自由阶段结束后剧情

**触发时机**：第一周结束

**反馈内容**（根据玩家表现动态生成）：

**成功路径**（安静观察 + 复盘 + 建设绘画室 + 获得画作）：
> **观察记录**：橙橙对安静 presence 不排斥。她进入新空间时会先检查桌面、光线和工具位置，这需要可预测性。她画的桌角、窗框不是"静物练习"，而是她在确认"世界是稳定的"。
> **所长评价**："你第一周没有急着让她'回应'你。这对橙橙来说，已经是很大的尊重了。"
> **获得画作**：《桌》
> **理解关键词**：秩序需求、安全感确认

**失败路径**（未解锁全部节点或未获得画作）：
**观察记录**：失败的观察记录。
**所长评价**：无。
**获得画作**：无。
弹出是否重试本周选项。

**关键设计**：
- 不是放在周结算，而是自由行动阶段后，周结算之前的一段剧情
- 即使第一周表现基础———是会失败的没那么高自由度。
- 所有路径都解锁后续周，但观察质量会影响后续解锁速度（这肯定不行，错过了就是错过了）

### 9.2 第一周画作详细设计

#### 《桌》
**画面描述**：
- 桌上物品特写，线条精细
- 阴影处理成熟，显示对光线的敏感
- 画纸质感：光滑，无褶皱

**叙事意义**：
- 橙橙关注的是"稳定的、不会动的"物体
- 桌角代表"边界"和"支撑"
- 理解关键词：**秩序需求**

**玩家如何获得**：
- 绘画室建成后，橙橙画的第一幅画
- 若玩家不贸然询问，橙橙会将画留在桌上（间接分享）

### 9.3 第一周关键条件与变量

#### 前置变量
- 无（第一周是起点）

#### 本周新增变量
| 变量 | 值 | 用途 |
|------|-----|------|
| `orange_first_contact` | `observe` / `interrupt` | 记录活动室初遇方式，决定所长复盘文本和是否需要补救 |
| `orange_first_contact_repaired` | true / false | 若初遇选错，玩家是否在所长复盘中获得并完成二次机会 |
| `painting_room_unlocked` | true / false | 初遇并完成所长复盘后解锁绘画室建设入口 |
| `painting_room_built_week1` | true / false | 第一周是否建成绘画室；决定能否进入节点4A |
| `week1_painting_followup_done` | true / false | 是否完成绘画室后续事件 |
| `week1_table_art_obtained` | true / false | 是否获得第一周关键画作《桌》 |
| `orange_favor` | 小幅增减 | 初遇、补救、绘画室后续共同影响橙橙好感 |
| `orange_artifacts` | `['桌']` 或空 | 记录第一周是否获得关键画作 |
| `week1_orange_result` | `success` / `fail` | 自由阶段结束后剧情的结算结果，决定是否弹出重试本周 |

#### 关键阈值
- 第一周存在硬性推进条件：必须完成“活动室初遇 → 所长复盘 → 建设绘画室 → 绘画室后续”链路，才算成功进入后续橙橙主线。
- 成功最低条件：`painting_room_built_week1 = true` 且 `week1_painting_followup_done = true` 且 `week1_table_art_obtained = true`。
- 初遇选错不会直接失败，但必须通过所长复盘中的补救机会修正，否则后续获得画作的条件更难或直接失败。
- 未建设绘画室视为第一周橙橙主线失败：不是“少看一点内容”，而是没有为橙橙建立必要的表达空间。

---

### 9.4 第一周后果设计

#### 立即反馈
- 活动室初遇：橙橙是否收起本子，直接反馈玩家靠近方式是否合适。
- 所长复盘：解释初遇中玩家没有看懂的细节；若初遇选错，提供一次二次机会。
- 建设绘画室：消耗资金，立即开放绘画室，并把剧情推进到节点4A。
- 绘画室后续：橙橙是否继续画、是否留下《桌》，直接反馈玩家是否尊重她的空间试探。

#### 延迟反馈
- 自由阶段结束后、周结算前触发第一周橙橙专项反馈。
- 成功时，反馈玩家已经初步看见橙橙的秩序需求和安全感确认方式。
- 失败时，不再用“观察质量较低”弱化后果，而是明确提示第一周橙橙主线没有完成，并弹出是否重试本周，可以不重拾，不影响照护所本身经营。
- 第二周不再根据“观察质量”软解锁；第一周失败意味着错过关键入口，需要重试或走后续缺失路线。

#### 解锁/关闭的后续入口
- 完成所长复盘后解锁：电脑中的绘画室建设。
- 建成绘画室后解锁：第一周绘画室后续事件。
- 获得《桌》后解锁：第二周橙橙画作主线的正常入口。
- 未建绘画室或未获得《桌》：第一周判定失败，后续橙橙剧情不按正常链路推进。

#### 获得或错过的画作/记录
- 获得：《桌》（第一周唯一关键画作）。
- 错过：《桌》即第一周失败，不设置《窗框》《重复角度》或“无题观察记录”等替代收藏。
- 初遇选错但完成复盘补救，仍可获得《桌》。
- 绘画室后续中贸然追问“你要画什么”，橙橙放下笔，不留下画作。

#### 对父亲、所长、机构数值的影响
- 父亲：第一周不引入
- 所长：完成复盘可小幅增加所长信任；初遇选错但愿意补救时，所长评价仍偏正向。
- 机构资金：建设绘画室必然减少资金，这是第一周理解橙橙所付出的现实成本。
- 机构口碑：无直接影响
- 橙橙好感：活动室初遇、复盘补救、绘画室后续共同影响；不设置一次性大幅奖励。

### 9.5 第一周不写的内容

- **不引入父亲正式沟通**：第一周聚焦橙橙与玩家的建立关系
- **不触发重大声音危机**：声音敏感是橙橙的特性，但第一周不设计"危机"
- **不让橙橙主动递画**：第一周橙橙不会主动分享，最多"留在桌上"
- **不把绘画室建成写成"橙橙立刻开始开心画画"**：第一周的重点是空间试探，而不是关系突破
- **不设计替代画作路线**：第一周关键画作只有《桌》，错过就是错过
- **不把未建设绘画室写成低收益路线**：未建设绘画室会中断第一周橙橙主线
- **不把失败包装成普通结算**：第一周失败需要明确提示，并提供重试本周选项
- **不取消补救**：初遇选错仍可通过所长复盘获得二次机会，但补救后仍需完成绘画室链路

## 10. 第二周详细设计：声音事件与专业认知

**本周结构类型**
危机处理链结构：突发事件 → 玩家即时处理 → 所长/同事解释专业原因 → 解锁道具/支持方式 → 后续房间互动验证玩家是否真正学会。

**本周核心问题**
玩家第一次面对ASD孩子的感官危机。学会：理解而非纠正，降噪而非解释。

**本周叙事目标**
- 让玩家亲身体会橙橙的声音敏感不是"胆小"，而是真实的生理痛苦
- 引入"感官支持"概念，替代"忍受"或"解释"
- 陈岚首次透露自己小时候也对某些声音极其敏感（背景线索开端）
- 解锁降噪耳机道具
- 让玩家理解：错误处理的代价是真实的（橙橙次日缺席）


*## 橙橙第3—7周简纲

第3周主线讲沟通替代 / 选择系统，橙橙支线就写她如何通过选择卡、纸张、材料表达“要/不要”。

第4周主线讲机构展示、儿童作品公开、边界与同意，橙橙支线就写她的画被认为适合展示，玩家如何征询她。

第5周主线讲家长沟通与家长介入，橙橙支线就写父亲来观察她，学习坐远一点、先看、不问。

第6周主线讲评估、支持档案、跨机构沟通，橙橙支线就写她不在照护所，玩家整理她的支持档案。

第7周主线讲变化支持、视觉流程、个别化支持计划，橙橙支线就写她回来以后，如何重新接住变化。

### 第3周：她怎么表达选择

这一周保留“选择系统”。

主线里引入心理学老师/儿童发展医生来照护所了解情况。她不是来治疗橙橙，而是提醒主角：低口语孩子不一定没有意见，很多表达藏在动作里。停笔、推开、盖住、转身、把东西放远，都可能是沟通。

橙橙线里，她遇到新画纸、新材料或新的摆放方式。她没有说“不”，只是停笔，把纸推远。玩家要学会给她两个可理解的选择，而不是直接替她安排好。

本周干预科普点：功能性沟通、替代沟通、选择卡。

本周画作：《两张纸》。
意义：橙橙开始把“我可以选择”放进自己的表达系统里。

---

### 第4周：她的画能不能被别人看见

这一周保留边界冲突，而且要作为中期重点。

照护所有一个公开展示、社区活动、宣传报道或赞助机会，橙橙的画被认为“很适合展示”。对机构来说，这是口碑和资源；对橙橙来说，这是她的表达被拿到公共空间里。

主线里，心理学老师/医生正好看到这件事，提醒主角：孩子的作品、照片、记录都不是单纯素材。尤其橙橙低口语，不能因为她没有明确说“不”，就默认她同意。

橙橙线里，玩家要决定怎么征询她。可以用第3周建立的选择卡，让她选择：展示或不展示，原画或复制件，署名或匿名，现在决定或明天决定。

成功路径：橙橙接受匿名复制件展示，原画留在自己的文件夹里。
失败路径：机构直接使用她的画，橙橙开始收画，本子合上，不再轻易把画留在桌上。

本周干预科普点：知情同意、儿童参与决策、表达边界。


---

### 第5周：爸爸看见了什么

这一周是家长开放观察日。父亲观察。

父亲来到照护所，想看看橙橙“到底有没有进步”。他不是反派，只是习惯用说话、服从、成果来判断孩子状态。他可能会说：“橙橙，画给爸爸看看。”橙橙停笔。

主线里，老师/医生提到家长介入。很多支持如果只发生在照护所，回家以后就会断掉。父亲需要学的不是一套理论，而是怎么坐、怎么等、什么时候不问。

橙橙线里，玩家要引导父亲先坐远一点，先看，不催，不要求橙橙表演。然后用前几周的画作解释：她画桌角不是没内容，她画耳机不是随便画，她选择纸张不是挑剔，她不递画也不等于不分享。

本周干预科普点：家长介入、亲子互动观察、家长心理教育。

---

### 第6周：橙橙不在照护所

这一周橙橙缺席。

橙橙父亲给她请了一周假，带她去儿童发展门诊做复评，也可能包含听觉敏感、职能治疗/感统相关评估。前几周父亲已经看到耳机、画作、选择卡和展示边界，他开始觉得“是不是该找更专业的人看一看”。这不是坏事，但也带来新的焦虑：如果医生说要更密集训练怎么办？如果别的机构说能让她“更听话”怎么办？

这一周橙橙本人不出现。绘画室里她的位置空着，桌上的纸还在，耳机也在原来的位置。主角第一次意识到：一个孩子不在的时候，她留下的秩序仍然需要被尊重。不能因为她没来，就把她的桌子借给别人用，把她的画夹挪走，或者把她的东西当公共材料。

主线里，老师/医生和陈岚可以讨论评估与干预的区别：评估不是为了给孩子贴标签，也不是为了证明谁对谁错，而是为了知道她需要怎样的支持。干预也不是把孩子训练成“看起来正常”，而是减少痛苦、增加表达、提高参与的可能性。

本周干预科普点：发展评估、职能治疗/感统评估、跨机构记录、支持档案。

### 第7周：她回来以后，怎么重新接住变化

橙橙从门诊复评回来，几天没来照护所，原本的节奏已经断了一次。对她来说，“重新回来”本身就是变化。

这一周可以安排一个小突发：绘画室因为维修、漏水、整理展示物，暂时不能按原样使用；或者房间还在，但桌子位置因为上周活动被移动过一点。这个变化不需要很大，越小越真实。橙橙回来后就在活动室画画。

玩家这时要把前面所有东西用起来：不能直接说“没事，和以前一样”。
要把变化画成视觉流程，让她知道：
你回来了。
画夹还在。
耳机还在。
原来的桌子有一点变化。
可以先检查。
如果不舒服，不想画也可以先坐一会儿。

第7周父亲也可以参与。他不再一上来问“今天怎么样”，而是把她的画夹放在固定位置，然后坐到门边等。

成功路径：橙橙没有马上恢复原状，但她愿意按照流程检查空间，带着画夹走过走廊，接受一个临时角落。最后她画下那条路。

本周干预科普点：结构化支持、视觉流程、变化预告、返场适应。

本周画作：《拐弯》或《回来》。

《拐弯》画的是走廊中间一个转角，转角处放着她的画夹，远处有一小块光。
《回来》可以画一张桌子，桌上有纸、耳机和选择卡，旁边空着一把椅子。

这一周的收尾不要写成大团圆。更适合写得轻一点：父亲没有完全懂，橙橙也没有突然变得亲近。但父亲知道了要先把东西放回固定位置；主角知道了变化要提前画出来；橙橙把画留在桌上，没有合上本子。
