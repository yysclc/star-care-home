# 收集系统使用说明（CG + 画作）

## 修改内容

### 1. 数据定义 (`src/data/collections.js`)
现在只包含 **两类** 收集品：

#### CG回忆（4个）
- `cg_office_first` - 初到照护所
- `cg_dream_entrance` - 梦境入口
- `cg_painting_room` - 绘画室时光
- `cg_outdoor_yard` - 户外小院

#### 孩子画作（4个）
- `artwork_orange_sun` - 橙色太阳（作者：小橙）
- `artwork_wood_ball` - 不规则圆球（作者：小木）
- `artwork_color_mix` - 颜色混合（作者：小橙）
- `artwork_clay_animal` - 粘土小动物（作者：小木）

### 2. 收集图鉴场景 (`src/scenes/CollectionScene.js`)
- 分类标签页：全部 / CG回忆 / 孩子画作
- 网格展示所有收集品
- 未获得的显示"？？？"
- 点击查看详情（描述、作者（画作类）、获取条件）

### 3. 入口
- **主页**：`StartScene` 的"收集图鉴"按钮 → 打开完整图鉴
- **OfficeScene（Hub系统）**：**左上角"第X周"下面**的"🎨 图鉴"按钮 → 快速打开图鉴

## 按钮位置说明

### OfficeScene 中的收集按钮
- **位置**：左上角，与"第X周"文本中心对齐（x=52, y=60）
- **样式**：矩形按钮，显示"🎨 图鉴"
- **交互**：悬停时高亮，点击打开收集图鉴场景

## 如何测试

### 方法1：使用测试脚本
1. 在浏览器中打开游戏
2. 按 F12 打开开发者工具
3. 在 Console 标签中粘贴 `test_collection.js` 中的内容
4. 运行测试命令：
   ```javascript
   testCollectAll()      // 收集所有物品（8个）
   testCollectSome()     // 收集部分物品（4个）
   testClearCollections() // 清除所有收集品
   testCheckProgress()    // 查看当前进度
   ```
5. 刷新页面，点击"收集图鉴"按钮查看效果

### 方法2：在游戏中触发收集
在游戏的适当位置调用 `collectItem()` 函数，例如：

```javascript
// 在序章完成后
import { collectItem } from '../data/collections.js';

// 收集 CG
collectItem(gs, 'cg_office_first');

// 在绘画室活动完成后
collectItem(gs, 'artwork_orange_sun');

// 在玩具室活动完成后
collectItem(gs, 'artwork_wood_ball');
```

## 如何扩展

### 1. 添加新的收集品
在 `src/data/collections.js` 中的 `COLLECTIONS_DATA` 对象添加新项：

```javascript
// 添加新的 CG
new_cg_id: {
  id: 'new_cg_id',
  category: 'cg',
  name: 'CG名称',
  description: 'CG描述...',
  obtainCondition: '获取条件说明',
  rarity: 'rare',  // 或 'common'
},

// 添加新的画作
new_artwork_id: {
  id: 'new_artwork_id',
  category: 'artwork',
  name: '画作名称',
  description: '画作描述...',
  obtainCondition: '获取条件说明',
  rarity: 'common',
  artist: '作者名字',  // 画作类需要指定作者
},
```

### 2. 在游戏中触发收集
在合适的剧情或活动完成后调用 `collectItem()`：

```javascript
import { collectItem } from '../data/collections.js';

// 示例：在完成序章后收集 CG
if (collectedCG) {
  const isNew = collectItem(gs, 'cg_office_first');
  if (isNew) {
    // 显示获得提示
    this.playStoryLines(['获得CG：初到照护所！']);
  }
}
```

### 3. 自定义图标
目前使用 emoji 作为图标。后续可以：
1. 在 `public/images/` 添加图标图片
2. 在 `CollectionScene.js` 中加载图片
3. 替换 emoji 文本为图片显示
